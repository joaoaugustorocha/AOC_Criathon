# Mega Computer Editon: Jogo Didático sobre Periféricos, Memória e Armazenamento

***

Este trabalho apresenta o desenvolvimento do jogo didático de tabuleiro Mega Computer Edition, criado para apoiar o ensino dos conteúdos de Periféricos de Entrada/Saída, Memória e Armazenamento na disciplina de Arquitetura e Organização de Computadores (AOC). O jogo utiliza desafios e palavras-chave para estimular raciocínio lógico, cooperação e fixação de conceitos de hardware. A metodologia inclui criação de cartas com diferentes níveis de dificuldade, definição de regras pedagógicas e testes práticos. O objetivo é tornar o aprendizado mais lúdico, interativo e conectado à prática.
